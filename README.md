# European Airports Analysis

## Overview
This is a brief exploratory data analysis of the European Airports Routes dataset. 
The goal of this small project is to gain insights into flight patterns, popular routes, and flight characteristics across Europe.

## Dataset
The dataset contains information about European flights, including departure and arrival airports, flight durations, prices, and weekly flight frequencies.

Dataset: https://www.kaggle.com/datasets/lunthu/european-airlines-routes/data

## Analyses Performed
- Top 10 departure and arrival airports
- Average flight duration
- Top 5 longest flight routes
- Top 5 most expensive flights
- Top 5 most popular routes by frequency

## Visualizations
- Bar charts for the busiest departure and arrival airports
- Line chart for the top 5 longest routes
- Bar chart for the top 5 most expensive flights (optional)
These visualizations should help to quickly understand typical flight characteristics across European airports.

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- GitHub for version control
